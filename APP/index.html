APP

发表于 2021-04-18 更新于 2021-05-16 分类于 清单

口袋里的生活

PC

  • Ccleaner–应该是PC最强的清理软件了吧
  • draw–轻量级绘图软件，功能强大，可以满足论文绘图要求–https://github.com/jgraph/drawio-desktop
  • Motrix–下载工具类似IDM
  • 五音助手–音乐下载软件
  • YesPlayMusic–一个UI很像Apple Music的网易云第三方客户端，Github获取
  • TaskbarTool–实现任务栏透明等等，Github获取
  • Mp3tag–编辑歌曲文件的封面等tag
  • Foobar–功能强大的本地音乐播放器
  • ShareX–功能强大的截图软件，如果不习惯QQ截图可以试试这个
  • Everything–文件检索，快速搜索电脑文件
  • 腾讯桌面整理–目前用的最好的桌面整理软件，包含搜索功能，可以取代Everything
  • 格式工厂–功能还是比较强大的，也没有好的替代品
  • Typora–文本编辑器，支持代码模式，界面简洁好看
  • VS code–代码编辑器，hexo全程使用VS code
  • wallpaper Engine–最强壁纸软件
  • AdGuard–广告拦截软件

Android

  • Poweramp–很棒的本地音乐播放器，已经在华为应用商店购买
  • 麻雀记–很棒的便签类软件，已经解锁高级会员，支持坚果云备份
  • 倒带–QQ音乐网易云二合一的音乐软件，可以同时登录两个账号
  • 厚墨–强大的开源阅读软件，订阅书源商店，书源多，界面美观，功能强大
  • 皮皮喵–开源漫画阅读软件，各种漫画都能搜到，确定是有广告
  • 存储空间清理–强大的空间清理软件，已解锁内购
  • 菜鸟教程–学习必备，虽然我并不怎么打开
  • Lithium–强大的本地阅读软件，支持各种格式
  • 贴吧Lite–第三方百度贴吧，冲着一键签到去的
  • MiXplore–最强的文件管理软件

Ios

  • EMMO和MOODA–这是两款日记软件，风格可爱，之前一直在电脑写日记，后面如果没有时间会现写在这里，每日心情很可爱，目前EMMO更符合国人用户习惯，功能也比较多一点
  • Things–很强大的时间规划软件，总体来说用着还行
  • 彩云天气Pro–天气软件，精确报时，分钟级降雨提醒
  • Documents–文件管理软件，功能相比自带的更加强大
  • 熊猫吃短信–短信拦截软件
  • Spark–之前的邮件一直用的是QQ邮箱，后面用TIM接收国内邮箱，国外邮箱就用Spark接收
  • CpuDasher–显示手机状态的桌面小部件，功能挺强
  • Shadowrocket–代理软件，Android和PC都使用是v2rayN,IOS用不了，只能使用这个
  • 蒙哥阅读器–这真的是一个很棒的中英对照阅读软件，很强大
  • 香色闺阁–虽然名字很那个，但是真的是很正经的开源阅读App，没有广告，界面也还可以，算是IOS里比较好的吧
  • 国家地理–美国国家地理杂志每日精选，图片都很棒
  • 小黑盒和酷安–一个是游戏社区，一个是数码社区，有兴趣可以下载
  • 钱迹–真的是很棒的记账App，我之前一直是在电脑进行记账，本以为手机记账又要重新开始，但是钱迹支持支付宝和微信的账单导入，真的很棒
  • Top Widgets–桌面小部件，支持更换软件图标，很不错
  • 快贴–用于实现手机和电脑的剪贴板共享
  • Kant–一个偶然发现的免费的习惯养成应用，无广告，界面简洁好看
  • WSH LST–可以添加appstore的应用到这里，同时能看到价格，我是当作app的收藏夹来用的
  • 小推–IOS少有的第三方微博，界面美观无内购
  • HW Editor–Hexo博客的手机客户端，但是我太菜了，没弄好
  • TestFlight–破解必备啊
